# Api

#### Como executar o projeto com Docker

- `$ yarn install`

- `$ yarn build`

- `$ docker-compose up -d`

##### Como executar o projeto localmente

- Configure seu arquivo .env utilizando como modelo o .env.example

- `$ yarn start:dev`
